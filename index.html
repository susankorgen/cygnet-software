Towers of Hanoi

Pure, raw JavaSript, HTML, CSS, and SVG.

The SVG graphics are not downloaded images; they are generated as SVG XML content when this code runs.

Showcases UX and UI development best practices independent of coding language and technology stack.

Animates the classic recursion problem.

Graphically illustrates the cost of order 2n computation.

Supports a user language choice of English or French and easy translation for other user languages.

A solid first implementation.

8 days, after 4 years full time using a different, proprietary server side language full time for database development. Prior to these 4 years, I wrote and generated JavaScript as a UI developer full time from 2009-2016.

The 8 days started with 2 days of Javascript: The Good Parts by Douglas Crockford. Page 34 mentions the hanoi problem and provides a loop. Recursion can be challenging to visualize.

Page 34 inspired me to build a hanoi animation as a JavaScript code example, while packing in as many UI/UX best practices from my past UI development experience as I could fit at this small scale.

Comments at the end of main.js list the specific planned updates to complete the example. These comments list standard UI/UX best practices I think this code could demonstrate, given time to add them.
